drat repository of Johannes Ranke

 

You can make use of this repository by using the drat package. Before installing anything, you should update your R packages already installed, e.g. by starting R, and pasting the command

update.packages()

or similar, depending on your R installation.

For installing e.g. gmkin or upgrading to the latest released version, please install the drat package (if not already installed)

install.packages("drat")

add my repository

drat::addRepo("jranke")

and install as usual

install.packages("gmkin")
